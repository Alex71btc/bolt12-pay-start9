# BOLT12 Pay – Start9 Service

Self-hosted Lightning payment and identity server with BOLT12 support, packaged for Start9.

## Features

* ⚡ BOLT12 Offers (via LNDK)
* 🔁 LNURL + Lightning Address fallback
* 🔐 Fully self-hosted
* 🌐 Optional Nostr integration

## Status

🚧 Work in progress – Start9 service packaging

## Requirements

* StartOS
* Bitcoin Core
* LND
